# useful-vpk-for-vita
psv上一些有趣的工具备份

## 简介 / Introduction

这是一个PlayStation Vita (PSV) 实用工具和自制软件的备份仓库。本仓库用于收集和组织各种有用的VPK包和插件。

This is a backup repository for useful tools and homebrew software for PlayStation Vita (PSV). This repository is used to collect and organize various useful VPK packages and plugins.

## 目录结构 / Directory Structure

```
.
├── tools/          # 系统工具 / System tools
├── plugins/        # 插件文件 / Plugin files
├── homebrew/       # 自制应用 / Homebrew applications
└── utilities/      # 实用工具 / Utility tools
```

## 使用说明 / Usage

1. 浏览相应的目录查找你需要的工具
2. 下载对应的VPK文件
3. 通过VitaShell或其他安装工具将VPK安装到你的PSV上

---

1. Browse the corresponding directory to find the tools you need
2. Download the corresponding VPK file
3. Install the VPK on your PSV through VitaShell or other installation tools

## 注意事项 / Notes

- 请确保你的PSV已经安装了HENkaku或h-encore等自制固件
- 安装任何第三方软件前请备份你的数据
- 本仓库仅用于备份和学习目的

---

- Make sure your PSV has HENkaku or h-encore installed
- Back up your data before installing any third-party software
- This repository is for backup and learning purposes only
